# Hotel
Sistema de reservación para hotel.

## Tecnologías

<ul>
  <li>Java</li>
  <li>Swing</li>
  <li>JDBC</li>
  <li>C3P0</li>
  <li>JCalendar</li>
  <li>Patrones de diseño(MVC, DAO y Factory Method)</li>
  <li>SOA</li>
  <li>MySql</li>
</ul>
